# wake2vec devlog 2026-03-31

## Mistral 7B P1 session 1 (fresh start)

new model, new account for Mistral 7B v0.3, sliding window attention, 32K base vocab. the architectural comparison we've been wanting. script: `Wake2vec_Mistral-7B-v0.3_P1.py`.

**vocab:** 32,768 base → 77,321 total (+44,553 Wake tokens). similar injection profile to TinyLlama (32K → 76.5K). much larger injection than Llama 3.2 models (128K base, only ~44K new).

**SEQ_LEN 512 → 256:** first attempt at SEQ_LEN 512 gave 163s/step which was slower than Qwen 14B (124s/step). the 44K new embeddings + AdamW optimizer states on a 7B model at SEQ_LEN 512 caused the T4 to thrash, so dropped to 256, which is fine for P1, embedding training only needs local context, not long-range patterns.
