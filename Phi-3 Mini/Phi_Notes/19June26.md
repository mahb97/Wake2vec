## Notes

### The methodological caveat, restated

val loss is not comparable across vocab-share models. Phi (58% share, vocab 76,511, random baseline 11.25) and Llama 3.2-3B (26% share, vocab 172,451, random baseline 12.06) measure different tasks: Phi's val is dominated by un-learned Wake (hard), 3B's by known English (easy). the 2x2 single-variable comparison (Phi vs 3B) is decided at **generation quality**, not P1 val. P1 val only tells us the embedding configuration is forming.

### a textbook bet

Phi is the only model in the lineup whose pretraining *data* is the controlled variable rather than architecture or scale. Microsoft's "textbook quality" filter is the maximal distance from Finnegans Wake's chaos. so the phi question is: **does knowing English properly make you better or worse at learning to break it?**

if Phi builds Wake as fast as TinyLlama and generates as densely, then it doesn't matter ss injection overrides pretraining bias. if Phi lags it means that clean priors resist deformation, thus Joyce needs a model that learned English messily.
