## Notes

3B P3 closes the third architecture loop. TinyLlama's P3 + P3b proved the auxiliary losses don't move when P2 has already implicitly solved morpheme composition. Llama 1B's P3 confirmed the null cross-architecture. 3B P3 tests whether scaling the base model (1B to 3B) changes anything when the tokenizer and Wake injection size stay constant. Prediction: it won't. The starting L_morph of 0.0006 is already at the floor for what 128K-vocab Llama can encode geometrically; strong lambdas should burn through 1000 steps without moving it.

if the prediction holds, can see a three-architecture sweep: 32K-vocab TinyLlama at 1.1B, 128K-vocab Llama at 1B, 128K-vocab Llama at 3B, all three null. resulting analysis: *morpheme-compositional structure in Finnegans Wake is fully encoded by the P2 LM objective alone, independent of base model scale or tokenizer size*.

the 8B run in parallel will eventually give the fourth datapoint at the top of the Llama scale ladder. that's the run that closes the family.
