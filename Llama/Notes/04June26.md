### Memory-safe analysis cell changes

analysis cell has been rewritten to survive on T4. main changes:

1. **PCA subsampled** to 10,000 base tokens (from full 128K) and n_components capped at 50 (from 100). PCA on the full base matrix was the kill point, sklearn's PCA memory is O(min(n,d)²); 10K × 3072 fits, 128K × 3072 doesn't.
2. **Drift computed in 20K chunks** instead of full-matrix broadcasts which avoids holding 128K × 3072 × 2 arrays simultaneously.
3. **Nearest neighbors refactored** to avoid the full E_unit copy so only normalises base, computes per-token.
4. **`gc.collect()` between every section** and explicit `del` for large intermediate arrays.
5. **Scatter plot subsampled** to 10K base points (matplotlib was choking on 172K).

PCA is now on a subsample, which is noted in the JSON. statistically valid for the variance estimates that matter.
